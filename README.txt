# Tenacity-Tutors-Admin
Is a mobile app platform that allow students to find a tutor of their desired subject.
The application consist of video calling feature which is integrated with Jitsi SDK.

The Admin
  *Sign up and login to the app.
  *Can send communication messages to students
  *Can able to assign a tutor to a student.
  *Can able to manage students and tutors
  *Can able to add and remove subjects
  *Send direct messages to students (Chat system)
  *Update profile
The Tutor
  *Sign up and login to the app.
  *Update profile
  *Accept request assigned to hime or her by the Admin.
  *Start a tutroeing using a video call.
  *Update the status of the tutoring
